Semiconductor Main Memory
หน่วยความจำหลักในปัจจุบันใช้ Semiconductor โดยข้างในจะมีหน่วยพื้นฐานคือ Memory Cell 
	Memory cell สามารถ "เขียน" และ "อ่าน" ได้โดยจะเก็บได้ 2 สถานะคือ 0,1

Semiconductor Memory แบ่งได้ 2 ประเภทหลักๆคือ
1.RAM(Volatile)
สามารถอ่านและเขียนได้ โดยเก็บข้อมูลเป็น 0,1 แต่เก็บข้อมูลแค่ชั่วคราวเมื่อไฟดับข้อมูลจะหายและแบ่งได้ 2 รูปแบบคือ
	1.DRAM จะเก็บข้อมูลในประจุไฟฟ้า , มีการ Refresh ตลอดเวลา และมีราคาถูกและความจุที่สูงทำให้มักถูกใช้เป็น Main Memory
	2.SRAM จะใช้วงจร Flip-Flop ในการเก็บข้อมูล , ไม่ต้อง Refresh และมีความเร็วที่สูงกว่าและแพงกว่า DRAM มักถูกใช้เป็น Cache
2.ROM(Non-Volatile)
	เน้นใช้อ่านและเก็บข้อมูลถาวร ไม่เหมาะแก่การเขียนข้อมูล มีหลายประเภทเช่น
	ROM : ข้อมูลที่ถูกบันทึกมาตั้งแต่ผลิต แก้ไม่ได้
	PROM : เขียนข้อมูลได้เพียงครั้งเดียว
	EPROM : ลบข้อมูลด้วยแสง UV และ เขียนใหม่ได้หลายครั้ง
	EEPROM : ลบและเขียนข้อมูลใหม่ด้วยสัญญาณไฟฟ้า โดยเลือกทำบาง Byte ได้
	Flash Memory : คล้าย EEPROM แต่ลบข้อมูลได้เร็วกว่าและมีความหนาแน่นของข้อมูลมากกว่า

โครงสร้างและการทำงาน (Organization & Chip Logic)
	การจัดเรียง : Memory Cell จะเรียงเป็น Row/Column คล้ายตารางเป็นช่องๆ ใน 1 ช่องเก็บข้อมูลได้ 1 bit
	การเข้าถึงข้อมูล : Semiconductor ใช้ Address ในการระบุตำแหน่ง มีสัญญาณควบคุมหลักคือ WE(เขียน),OE(อ่าน)
	DRAM จะส่ง Address 2 รอบโดยครั้งแรกจะเลือก แถว(RAS)ก่อนแล้วเลือก คอลัมน์(CAS)ต่อเพื่อลดจำนวนขาช่วยให้ประหยัดต้นทุน
	DRAM มีการ Refresh ข้อมูลตลอดเวลาเพื่อกันไม่ให้ข้อมูลเพี้ยน

แนวคิดที่สำคัญ (The Trade-off)
	ไม่มีหน่วยความจำที่ดีที่สุดทุกด้าน แต่ต้องแลกกันมาระหว่าง ความเร็ว(Speed),ราคา(Price),ความจุ(Capacity)	
	SRAM : มีความเร็วที่สูงแต่มีราคาที่แพง (เหมาะกับ Cache)
	DRAM : มีความเร็วที่ช้าแต่มีความจุสูง และมีราคาที่ถูก (เหมาะกับ Main Memory)
	ROM : เหมาะแก่การเก็บข้อมูลถาวร
